# angular-2-practices

#### Requrements ####
`node`>= `4.x.x`</br>
`npm` >= `3.x.x`,

#### Steps####
First run `npm install -g angular-cli` and then follow the following steps:
`1` Clone Repository,
`2` Run `cd angular-2-practices`,
`3` Run `npm install`,
`4` Run `ng-serve`,
`5` Go to Browser and redirect to `localhost:4200`,
